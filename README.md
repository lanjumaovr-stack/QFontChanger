# QFontChanger

QFun Script - Replace QQ fonts globally with custom TTF/OTF fonts.

## Features
- Local font scanning and switching
- Real-time font preview
- Lag character filtering (auto fallback to system font for missing glyphs)
- Long text auto-truncation (200 char threshold)
- Malicious stacked invisible character detection and hiding
- Tutorial dialog with font download site link

## Usage
1. Place .ttf/.otf font files into the script directory
2. Tap the floating menu item in QFun to open font selector
3. Select a font and tap Apply
